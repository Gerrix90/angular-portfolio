Core Framework

Overview

CoreFramework is a php web application framework that I build in my free-time. This was a self undertaking to understand php little better and also to understand what goes into an MVC framework. This was an extensive project which took me almost 2 years, to get to a usable and performing version after multiple rewrites. While not the fastest framework out there, the current version 3.0 is completely usable and gives performance of ~350 requests/sec * .
